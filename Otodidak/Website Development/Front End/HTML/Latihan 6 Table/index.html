Catatan

    Tag Table
    Merupakan elemen HTML untuk menampilkan data tabular(berbentuk tabel) untuk membuat table dalam HTML menggunakan tag 
 

    Struktur tabel
    Tabel terdiri atas struktur kolom, baris, cell, header

    Baris merupakan bagian tabel yang berada sejajar secara horizontal untuk membuat baris bisa menggunakan tag table row  di dalam tag table atau tag table body atau tag table head atau tag table foot

    Kolom merupakan bagian tabel yang berada sejajar secara vertical untuk membuat baris bisa menggunakan tag table data  di dalam tag table row

	Cell atau data merupakan perpotongan antara baris dan kolom

    Header merupakan penanda atau judul pada baris atau kolom header juga bisa ditempatkan di keduanya(baris dan kolom) untuk membuat header menggunakan tag table header bisa disimpan di dalam tag table row ataupun tag table data

    Best practice struktur tabel
    Untuk membuat struktur table yang baik yang bisa dibaca oleh mesin pencari dan accessibility yang baik adalah dengan membuat struktur table yang menggunakan pembagian table head, table body dan table foot

    Attribute element table
    Terdapat beberapa attribute pada element table yaitu border, cellspacing, cell padding, colspan, rowspan

    •	border
    merupakan attribute untuk menambahkan garis disekitar cell diisi dengan angka satuanya pixel
    •	cell spacing
    merupakan attribute untuk menambahkan jarak disekitar cell 
    •	cell padding
    merupakan attribute untuk menambahkan luas di dalam cell 
    •	colspan
    merupakan attribute tag td untuk melakukan merging atau penggabungan beberapa cell dalam kolom
    •	rowspan
    merupakan attribute tag td untuk melakukan merging atau penggabungan beberapa cell dalam baris dan apabila sebuah td diberi attribute rowspan maka td tersebut akan menempati baris sesuai jumlah rowspan
