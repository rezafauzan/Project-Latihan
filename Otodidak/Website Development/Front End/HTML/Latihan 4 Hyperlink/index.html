Catatan

        Tag Hyperlink
        Link/Hyperlink/WebLink merupakan sebuah koneksi dari sebuah sumber web ke sumber lain
        Link pada HTML dibuat menggunakan tag  atau disebut dengan Anchor dan tag ini memerlukan satu attribute wajib yaitu href diisi dengan sumber tujuan
        
        Ada beberapa tujuan yang bisa menjadi sumber dalam tag Anchor ini yaitu :

    •	External Link
        Yaitu link yang mengarahkan pada halaman diluar website

    •	Internal Link / Relative URL
        Yaitu sebuah link untuk mengakses sumber dari domain yang sama dengan Website yang dibuat
    
        Disebut Relative URL karena bisa mengakses sumber dalam satu domain dengan berbeda folder
        
        Untuk mengakses sumber dengan beda folder bisa menggunakan format alamat 
        / = root 
        /namaSumber.extensi
        /namaFolder/namaSumber.extensi
        Dan apabila berada satu level diatas folder sekarang bisa menggunakan 
        ../namaSumber.extensi 
        ../namaFolder/namaSumber.extensi 
        Titik dua kali menyatakan bahwa akan keluar/naik satu kali dari folder sekarang

    •	Page Anchor 

        Merupakan sebuah hyperlink untuk mengakses sebuah halaman dengan bagian didalam halamanya
        Untuk bisa menggunakan Teknik ini diperlukan attribute id untuk diakses melalui href contoh
        Apabila hyperlink ini diklik maka akan langsung membuka halaman kemudian menuju ke section dengan id yang ditulis di href
    
    Attribute Target

    Attribute target ini merupakan sebuah attribute untuk menentukan sifat hyperlink ketika diklik

    •	_self
        _self merupakan value default akan membuka tujuan pada tab yang sama
        Apabila tag Anchor tidak memiliki attribute target maka value target akan menggunakan _self

    •	_blank
        blank merupakan value target yang akan membuka tujuan pada tab baru

    •	_parent
        _parrent merupakan value target secara viewport

    •	_top
        _parrent merupakan value target secara viewport
