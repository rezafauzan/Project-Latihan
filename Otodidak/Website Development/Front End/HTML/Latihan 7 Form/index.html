Catatan

        Form
        Merupakan sebuah element HTML untuk mengelola input yang dilakukan oleh user baik ketika mengisi sebuah kolom input maupun sebuah pilihan pada combo box, dropdown atau pun radio button atau sebuah tombol 
        Contoh form seperti form login, form pendaftaran, dll
        
        Untuk membuat sebuah formulir dibutuhkan tag form 
 untuk membungkus satu kumpulan elemen-elemen form agar kemudian nilai dari input user bisa ditampung atau digunakan
        
        Komponen Form 
        Input pada form bisa berbentuk macam macam yaitu :

        •	Input / Text Field
        
            Kolom input berupa text atau pun number biasanya disebut dengan text field untuk membuat field ini menggunakan tag  dan tag ini memiliki attribute type berfungsi untuk menyaring input user sesuai type field nya 
            Salah satu Value attribute type ini adalah :
            
                	Text
                Ini merupakan value default dapat menampung string apapun
                	Number
                Untuk membatasi bahwa field hanya bisa diisi dengan angka dan huruf e sebagai eksponen input ini juga memiliki tombol tambahan untuk increment dan decrement
                	Tel
                Untuk membatasi bahwa field ini hanya bisa diisi angka dan lebih spesifik digunakan untuk menulis nomor telepon dan bisa juga diberikan pola
                	Password
                Sama seperti text namun apapun yang ditulis di dalam field ini akan ditampilkan sebagai bullet ini bukanlah enkripsi jadi apapun yang diisikan masih bisa dilihat
                	Email
                Input type email bisa diisi dengan string apapun namun harus menyertakan @(wajib) dan domain(optional)
                Dan masih banyak lagi type type inputan seperti color, date, dll

            Selain Attribute type field ini juga memiliki attribute placeholder yang akan memunculkan text sebagai hint di dalam field

        •	Text Area
            Merupakan kolom input mirip dengan tag input akan tetapi lebih besar biasanya digunakan untuk membuat kolom komentar atau pesan untuk membuat field ini menggunakan tag
