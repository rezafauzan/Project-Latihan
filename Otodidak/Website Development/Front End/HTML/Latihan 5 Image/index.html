Catatan

    Tag Image
    Yaitu merupakan tag untuk menampilkan gambar pada halaman website menggunakan tag  dan memiliki attribute src untuk membuat rujukan sumber yang akan ditampilkan
    Attribute src ini bisa merujuk pada internal resource ( berada 1 folder/domain dengan website ) atau external resourse ( dari website lain ) dan untuk sumbernya mesti ditulis beserta extensinya

    Akan tetapi penggunaan external resource memiliki beberapa resiko yaitu bisa terkena hak cipta, menggunakan bandwith pemiliki resource, dan beresiko apabila gambar di sumber asli hilang atau tidak dapat diakses akan merusak penampilan website karena gambar akan diisi dengan icon
    Attribute Tag Image

    Tag image memiliki attribute attribute yang dapat digunakan salah satunya adalah :

    •	src
    Merupakan attribute utama yang berfungsi untuk merujuk sumber yang akan ditampilkan

    •	alt
    Merupakan attribute untuk menampilkan text sebagai alternative apabila sumber tidak dapat ditemukan

    •	width
    Merupakan attribute untuk mengatur lebar dari elemen gambar yang ditampilkan apabila diset hanya widthnya saja maka height akan proporsi terhadap width

    •	height
    Merupakan attribute untuk mengatur tinggi elemen gambar yang ditampilkan apabila diset hanya heightnya saja maka width akan proporsi terhadap height

    •	title
    Merupakan attribute judul pada tag img muncul ketika dihover menggunakan cursor dan muncul juga ketika sumber hilang tapi tidak ada attribute alt

    Hyperlink + Image
    Didalam HTML memungkinkan untuk membuat image sebagai hyperlink dengan cara mengisi konten tag anchor menggunakan image
    
    Ini akan membuat image bisa diklik dan mengarah ke suatu tujuan sebagaimana hyperlink biasa
