Penggunaan Program Angkot

Terdapat 5 instansi angkot yaitu : angkot1, angkot2, angkot3, angkot4, angkot5

Masing masing instansi memiliki properti : Kas, Nama Supir, Trayek, dan Nama-nama Penumpang

Dan masing masing instansi memiliki method : penumpangNaik( namaPenumpang, daftarPenumpang ), dan penumpangTurun( namaPenumpang, daftarPenumpang )

method penumpang naik memiliki rules sebagai berikut :

  1. apabila angkot kosong maka penumpang akan duduk di kursi pertama (index 0)
  2. apabila angkot ada kursi kosong maka penumpang akan duduk di kursi yang kosong terlebih dahulu (index yang memiliki nilai undefined)
  3. apabila nama penumpang sama maka tidak bisa naik pada angkot yang sama
  4. ketika turun kursi akan di set sebagai undefined
  5. ketika turun penumpang akan membayar tarif sesuai trayek sebagai berikut :
    • Soreang - Bandung  : 8000
    • Soreang - Banjaran : 7000
    • Soreang - Ciwidey  : 7500
    • Soreang - Cimahi   : 7500
    • Soreang - Cililin  : 6500
    • Jauh Dekat         : 10000
