Control Flow

        Control Flow 
        Ada yang disebut dengan normal flow yang artinya program memiliki lebih dari satu statement maka statement tersebut akan di eksekusi dari atas ke bawah atau dari kiri ke kanan 
        Dan terkadang sebuah program memerlukan untuk melewati baris program tertentu dan ini akan merubah alur kerja programnya biasanya ini dilakukan oleh Control Flow bisa menggunakan pengulangan( loop / iteration ) dan pengkondisian
        
        Pengulangan
        
        Pengulangan adalah sebuah konsep untuk mengerjakan atau mengeksekusi statement yang sama lebih dari satu kali sampai selesai baru kemudian lanjut ke statement berikutnya
        Ada banyak pengulangan di dalam javascript salah satunya adalah for, while, do while
        
        Contoh : Lihat Catatan.docx

        •	While
                Syntax : 
                while(kondisi){
                //Aksi / statement
                }
                Dan selama kondisi bernilai true lakukan aksi dalam blok akan dieksekusi atau akan dijalankan
            
                Apabila while loop tidak diberikan nilai false maka akan terus melakukan aksi tanpa berhenti yang disebut dengan infinity loop dan untuk menghentikan looping ini bisa oleh user dengan cara menutup halamannya atau bisa dengan memberikan statement untuk merubah kondisi menjadi false contoh
            
                Atau bisa juga dengan memberikan kondisi teriminasi seperti berikut
            
                Catatan : Kondisi terminasi adalah kondisi untuk menghentikan pengulangan
        •	For
                Syntax : 
                For(nilaiAwal ; kondisiTerminasi  ;  increment/decrement ){
                // Aksi / Statement
                }   
                Mirip seperti while loop akan tetapi penulisanya lebih rapih karena nilai awal,kondisi terminasi dan increment/decrement ditulis di dalam parameter for()
                Tabel Penelusuran (Trace Table / Dry-Run Test )
                Tabel Penelusuran merupakan alat bantu untuk melihat setiap perubahan variable yang terjadi di dalam pengulangan
            
    Pengkondisian
    Pengkondisian atau disebut juga percabangan adalah sebuah keadaan dimana program bisa memilih treatment yang berbeda berdasarkan kondisi yang diberikan ada beberapa percabangan yang bisa dipakai salah satu nya adalah if, if else, if else if else, switch
