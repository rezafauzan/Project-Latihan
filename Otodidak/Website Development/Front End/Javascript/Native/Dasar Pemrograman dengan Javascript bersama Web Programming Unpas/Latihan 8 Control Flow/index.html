Control Flow

        Control Flow 
        Ada yang disebut dengan normal flow yang artinya program memiliki lebih dari satu statement maka statement tersebut akan di eksekusi dari atas ke bawah atau dari kiri ke kanan 
        Dan terkadang sebuah program memerlukan untuk melewati baris program tertentu dan ini akan merubah alur kerja programnya biasanya ini dilakukan oleh Control Flow bisa menggunakan pengulangan( loop / iteration ) dan pengkondisian
        
        Pengulangan
        
        Pengulangan adalah sebuah konsep untuk mengerjakan atau mengeksekusi statement yang sama lebih dari satu kali sampai selesai baru kemudian lanjut ke statement berikutnya
        Ada banyak pengulangan di dalam javascript salah satunya adalah for, while, do while
        
        Contoh : Lihat Catatan.docx

        •	While
                Syntax : 
                while(kondisi){
                //Aksi / statement
                }
                Dan selama kondisi bernilai true lakukan aksi dalam blok akan dieksekusi atau akan dijalankan
            
                Apabila while loop tidak diberikan nilai false maka akan terus melakukan aksi tanpa berhenti yang disebut dengan infinity loop dan untuk menghentikan looping ini bisa oleh user dengan cara menutup halamannya atau bisa dengan memberikan statement untuk merubah kondisi menjadi false contoh
            
                Atau bisa juga dengan memberikan kondisi teriminasi seperti berikut
            
                Catatan : Kondisi terminasi adalah kondisi untuk menghentikan pengulangan
        •	For
                Syntax : 
                For(nilaiAwal ; kondisiTerminasi  ;  increment/decrement ){
                // Aksi / Statement
                }   
                Mirip seperti while loop akan tetapi penulisanya lebih rapih karena nilai awal,kondisi terminasi dan increment/decrement ditulis di dalam parameter for()
                Tabel Penelusuran (Trace Table / Dry-Run Test )
                Tabel Penelusuran merupakan alat bantu untuk melihat setiap perubahan variable yang terjadi di dalam pengulangan
            
        Pengkondisian
        Pengkondisian atau disebut juga percabangan adalah sebuah keadaan dimana program bisa memilih treatment yang berbeda berdasarkan kondisi yang diberikan ada beberapa percabangan yang bisa dipakai salah satu nya adalah if, if else, if else if else, switch
            •	If
                Syntax : If(Kondisi){
                    Aksi/Statement
                }
                Statement akan dijalankan apabila kondisi menghasilkan true dan akan keluar dari bloknya apabila kondisi bernilai false
                Penulisan pengkondisian dengan if ini bisa menangkap lebih banyak pengkondisian dengan menggunakan beberapa if atau bisa juga menggunakan else 
                Atau 
            •	If else
                Syntax: if(kondisi){
                    Statement/Aksi
                } else {
                    Statement/Aksi
                }
                Akan tetapi contoh if else diatas kurang baik karena apabila variable angka diisi selain angka maka akan bernilai false sehingga akan menjalankan statement pada blok else
                Untuk mengatasi permasalahan pada Source Code diatas dibutuhkan pengkondisian berlanjut menggunakan syntax Else If                
                •	Else If
                    Syntax :
                    If( kondisi1 ){
                        Aksi/Statement
                    } else If ( kondisi2 ){
                        Aksi/Statement
                    }
                    else{
                        Aksi/Statement
                    }
                    Else if ini merupakan percabangan untuk menambahkan kondisi yang berbeda dan else if disimpan diantara if dan else dan jumlah else if bisa banyak 
                •	Switch
                    Syntax : 
                        switch(ekspresi){
                            case 1 :
                                statement/aksi
                            break ;
                            case 9 :
                                statement/aksi
                            break ;
                            case 4 :
                                statement/aksi
                            break ;
                            case 5 :
                                statement/aksi
                            break ;
                            case n :
                                statement/aksi
                            break ;
                            default:
                                statement/aksi
                            break;
                        }
                    Switch merupakan percabangan dengan ekspresi sebagai parameternya dan menjadikan nilai ekspresi sebagai kondisinya yang disimpan pada case dan apabila berada diluar case maka statement didalam default akan dijalankan
                    Dan khusus pada kasus ini dimana yang menjadi case merupakan integer atau bilangan bulat maka harus nilai variable angka akan dipaksa menjadi integer menggunakan fungsi built in javascript yaitu parseInt() atau merubah nilai case menjadi string angka
                    
                    Contoh lain switch bisa dilihat pada catatan.docx
                    
                    Apabila case pada switch memiliki statement sama , beberapa case bisa digabungkan dan cukup pada block case terakhir diisi dengan statement
                    Syntax : 
                    switch(ekspresi){
                        case 1 :
                        case 9 :
                        case 4 :
                        case 5 :
                            statement/aksi
                        break ;
                        case n :
                            statement/aksi
                        break ;
                        default:
                            statement/aksi
                        break;
                    }
