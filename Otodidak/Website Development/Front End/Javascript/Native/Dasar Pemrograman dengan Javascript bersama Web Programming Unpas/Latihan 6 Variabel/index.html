Variable pada javascript

        Apa itu Variabel?
        
        Variable adalah sebuah tempat atau wadah yang memiliki nama(Identifier) yang digunakan untuk menyimpan nilai
        Ruang lingkup variable meliputi Deklarasi, Inisialisasi, Assignment yaitu :
        
        •	Deklarasi
        Yaitu tahap untuk mendaftarkan variable kedalam lingkup yang sesuai atau spesifik
        Deklarasi dalam javascript bisa menggunakan var, let, const
        
        Contoh : var 
        
        Nama variable bebas dibuat atau ditulis apapun namun Ada aturan aturan dalam membuat nama variable yaitu :
            1.	Tidak boleh menggunakan spasi
            2.	Tidak boleh diawali angka
            3.	Boleh menggunakan angka selain diawal
            4.	Tidak boleh menggunakan keyword dan reserved word
                Yaitu kata kata yang merupakan bagian syntax javascript 
        
        Camel case merupakan format penulisan yang paling sering digunakan yaitu dengan menuliskan kata awal dengan huruf kecil dan kata selanjutnya menggunakan huruf besar di awal 
        
        Untuk mendeklarasikan banyak variable bisa juga menggunakan shorthand 
        
        Dan didalam Variable ada yang disebut dengan Variable Scope 

        •	Inisialisasi
        Yaitu proses untuk pemesanan atau penyediaan memory untuk variable
        
        •	Assignment
        Yaitu tahapan untuk menetapkan nilai yang spesifik ke dalam variable
