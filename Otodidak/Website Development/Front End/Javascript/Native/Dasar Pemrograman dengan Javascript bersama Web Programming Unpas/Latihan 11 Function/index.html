    Function
    Function Merupakan sebuah sub-program atau sub-routine yang dapat dipanggil di bagian lain pada program atau lebih singkatnya function itu merupakan program didalam program
    Function juga disebut sebagai prosedur (Kumpulan statement untuk melakukan tugas atau menghitung/mengkalkulasi nilai)
    Untuk Menggunakan function ,function harus dibuat terlebih dahulu lalu memanggilnya
    Function termasuk kedalam first class object 
    Dan function merupakan struktur dasar pembentuk javascript (Fundamental Building Block)
    
    Function di javascript bisa dibuat sebagai method,class,constructor dan module oleh sebab itu javascript menjadi sangat powerfull

    Parameter Dan Argument
    Parameter adalah variable yang ditulis didalam kurung pada saat function dibuat digunakan untuk menampung nilai yang dikirim saat function dipanggil
    Argument adalah nilai yang dikirimkan ke parameter saat fungsi dipanggil 
    
    Parameter dan Argument bisa dianalogikan sebagai bahan atau material untuk diproses lebih lanjut di dalam sebuah fungsi
    Argument juga bisa diisi dengan fungsi yang mengembalikan nilai dimana nilai hasil dari fungsi tersebutlah yang akan menjadi argument
    di dalam javascript apabila argument lebih banyak dari parameter maka kelebihanya akan diabaikan
    Sedangkan apabila parameter lebih banyak maka kelebihanya akan diisi nilai default yaitu undefined

    Arguments 
    Arguments atau pseudo variable Merupakan array yang berisi nilai nilai  yang dikirimkan saat fungsi dipanggil 
    Setiap argument yang dikirim ke parameter akan dikirim juga kedalam pseudo variable arguments bertipe array dan memiliki index

    Refactoring
    Refactoring adalah sebuah proses mengubah code agar menjadi lebih baik tanpa mengubah fungsionalitasnya
    Refactoring ini bermanfaat untuk meningkatkan 
        •	Readability yang dimana sebuah Source Code itu harus mudah dibaca untuk pembuat maupun orang lain , juga bermanfaat agar Source Code menjadi 
        •	DRY (Do Not Repeat Yourself) berkaitan dengan duplikasi kode contohnya penggunaan variable secara berlebihan effisiensi penggunaan function,method dll
        •	Testabillity
            Yaitu penulisan code agar mudah dilakukan pengujian
        •	Performance
            Yaitu Cara penulisan code supaya memiliki performa yang optimal contohnya penggunaan looping yang baik atau memilih tipe data yang sesuai dan alokasi memory yang efektif
        •	Maintainabillity
            Yaitu agar Program yang dibuat mudah dikelola dan dikembangkan
