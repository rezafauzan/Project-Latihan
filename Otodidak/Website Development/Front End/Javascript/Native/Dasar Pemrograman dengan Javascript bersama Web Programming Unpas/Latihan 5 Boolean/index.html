Nilai dan Tipe Data Pada Javascript : Boolean

        •	Boolean
            Yaitu sebuah tipe data yang digunakan untuk merpresentasikan logika true atau false dan Boolean ini sudah ada sejak lama dan disemua Bahasa , Boolean ini diciptakan oleh George Bool pada tahun 1954 beliau adalah seorang ahli matematika bidang logika 

            Boolean ini biasanya digunakan pada statement pengkondisian untuk menentukan aksi yang berbeda dan alur kendali program

            Boolean ini memiliki 2 nilai yaitu true dan false contohnya 
            Let x = 20

            (20\>10) maka akan menghasilkan nilai true 

            Cara lain untuk menghasilkan Boolean di javascript adalah dengan fungsi Boolean(nilai)
            Contoh Boolean(20<\10) ini akan menghasilkan nilai false

            Fungsi ini juga bisa hanya satu nilai seperti Boolean(10) untuk menghasilkan true

            Dan untuk menghasilkan false bisa memasukan nilai falsy seperti pada tabel pada catatan.docx

        •	Undefined
            Dihasilkan ketika mendeklarasikan variable tapi belum mengisikan nilainya contohnya 
            Let x ; akan menghasilkan undefined

        •	Null 
            Adalah sebuah nilai kosong yang bisa diberikan pada variabel
