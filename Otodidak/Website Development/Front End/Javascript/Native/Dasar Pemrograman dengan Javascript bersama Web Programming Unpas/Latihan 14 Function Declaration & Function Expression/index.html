        Function Declaration Vs Expression
        •	Function Declaration
            Syntax : function identifier (parameter [Optional]){function body}
            
            Function declration ini cenderung lebih flexible karena bisa dipanggil setelah function ditulis ataupun sebelum function ditulis ini dikarenakan javascript memiliki sesuatu yang dinamakan hoisting maka pendeklarasian fungsi ini akan terlebih dahulu disimpan kedalam memory sebelum di Source Code Di Eksekusi 
        
        •	Function Expression
            Syntax : function identifier [Optional] (parameter [Optional]){function body}
            
            Disebut function expression karena function disimpan di dalam variable, untuk function expression ini harus di definisikan terlebih dahulu tidak bisa dipanggil sebelum didefinisikan seperti function declaration 
            Akan tetapi function expression lebih powerfull karena bisa digunakan sebagai closure, sebagai argument untuk function lain dan sebagai IIFE(Immediately Invoked Function Expression)
            
            Konsep function expression ini digunakan di javascript lanjutan dan javascript modern termasuk functional programming atau pada framework seperti Jquery
