    Membuat Object
    Ada beberapa cara untuk membuat object pada javascript yaitu dengan cara Object Literal,Function Declaration,Constructor Function( keyword new), Object.create()
    •	Object Literal
        let siswa = {
            nama : 'Reza',
            nip  : '0405',
            jurusan : 'Rekayasa Perangkat Lunak'
        }

        let siswa2 = {
            nama : 'Denita',
            nip  : '0504',
            jurusan : 'Rekayasa Perangkat Lunak'
        }
    Untuk membuat object literal cukup dengan membuat variable yang diisi dengan kurung kurawal 
    Kekurangannya adalah apabila ingin membuat object baru maka harus dibuat semua property dan method nya 

    •	Function Declaration
    Yaitu membuat object dengan cara membuat function untuk mengisi object 
    Contoh :
        // Function Declaration
        function itemBaru(namaItem,idItem){
            let item = {}
            item.namaItem,
            item.idItem
            return item
        }

        let item = itemBaru('Sendok',1)

    Lebih ringkas dari object literal karena templatenya dibuat sekali dan bisa membuat object berkali kali

    •	Constructor Function( keyword new)
    Mirip dengan function declaration dan constructor ini adalah function khusus untuk membuat object
    Penulisannya pun sama seperti function declaration dan ada kebiasaan untuk membedakanya yaitu dengan menulis nama function diawali huruf besar
    Dan dengan constructor ini tidak perlu menulis property dan object seperti pada function declaration melainkan cukup menggunakan syntax this.namaProperty
    Dan untuk membuat objectnya yaitu dengan cara membuat variable baru diisi dengan keyword new dan nama constructornya dan diisi parameternya
    Dengan menggunakan keyword new maka secara otomatis bisa dianggap akan dibuatkan variable this = {} dan mereturnnya 
    Namun apabila tidak diikuti keyword new akan dianggap pemanggilan function declaration yang dimana karena tidak ada variable object dan returnya maka akan mengembalikan nilai undefined
        // Constructor
        function Buku(namaBuku,idBuku){
            // bisa dianggap terdapat let this = {}
            this.namaBuku = namaBuku
            this.idBuku = idBuku
        }

        let buku = new Buku('Catatan.docx',2)
        // let buku = Buku('Catatan.docx',2) akan dianggap function declaration dan mengembalikan nilai undefined karena tidak ada object pada function Buku
