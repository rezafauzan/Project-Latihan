        Variable Scope
    Variable Scope Atau lingkup variable adalah bagaimana variable dapat diakses di dalam sebuah program ada dua jenis scope atau dua jenis lingkup yaitu block scope dan function scope
    
    •	Block Scope
    Variable yang berada di dalam block hanya bisa digunakan di dalam block tidak bisa digunakan di luar block  
    •	Function Scope

    Scope
    Secara umum ada beberapa lingkup variable yaitu Global Scope/Window Scope dan Local Scope 
    •	Global Scope
    Yaitu Variable Yang Disimpan Di Paling Terluar(berada pada Object Window) Source Code dan bisa digunakan dimana pun di programnya
    
    •	Function Scope
    Yaitu Variable yang dibuat di dalam block function hanya bisa diakses di dalam function itu saja,
