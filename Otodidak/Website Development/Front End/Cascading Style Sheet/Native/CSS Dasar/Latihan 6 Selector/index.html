        Selector

        Merupakan sebuah identifier sebuah element html yang akan diberi style, ada beberapa identifier yang bisa digunakan sebagai selector yaitu nama tag atau nama element HTML, id, class dan selector gabungan atau bisa disebut selector kompleks

        •	Nama Element HTML Sebagai Selector
            Nama dari element HTML bisa digunakan sebagai selector
        
        •	Id Sebagai Selector
            Memberikan attribute id pada element html value id tersebut bisa dijadikan sebagai selector pada css, pemanggilan id sebagai selector diawali dengan tanda “#”
            
            Dalam satu halaman html hanya diperbolehkan hanya ada 1 element dengan nama id tersebut tidak boleh ada element lain dengan id sama
            Element yang memiliki id dapat dijadikan value pada attribute href di tag anchor
            Id ini juga bisa digunakan di javascript
            Untuk css lebih baik menggunakan class

        •	Class Sebagai Selector
        
            Memberikan attribute id pada element html value id tersebut bisa dijadikan sebagai selector pada css, pemanggilan id sebagai selector diawali dengan tanda “.”
            Satu element html bisa memiliki banyak class dan satu class bisa dimiliki oleh banyak element
            Untuk memberi lebih dari satu class pada element html cukup dipisahkan dengan menggunakan spasi pada attribute class dan untuk css ditulis berdampingan menggunakan titik tiap nama class

        •	Selector Kompleks
            •	Gabungan Nama Element HTML, Id, Class Sebagai Selector 
            
            •	Multiple Selector
                Untuk memberikan styling pada beberapa selector bisa menggunakan koma sebagai separator untuk melakukan styling pada element element tersebut namun apabila ingin melakukan penambahan pada element spesifik elemen tersebut bisa dilakukan styling lagi 
        
            •	Spesifik Selector
                Untuk melakukan styling yang spesifik seperti tag a di dalam tag ul dan li seperti diatas diperlukan selector spesifik agar hanya tag a di dalam ul li tersebut yang dimodifikasi 
                untuk melakukan hal tersebut bisa menggunakan spasi sebagai separator contoh :
                
                Selector diatas artinya akan mencari tag a di dalam tag li yang berada di dalam tag ul sehingga hanya tag a yang ada di dalam ul li yang akan berubah
