Catatan

        CSS Pseudo Class
        Merupakan kelas semu dari sebuah elemen html untuk mendefinisikan style pada keadaan tertentu seperti ketika diklik ketika cursor berada diatas element dll
        Disebut kelas semu karena kelas ini ada atau aktif hanya pada kondisi tertentu
        •	Pseudo Class Untuk Element dan Element Link
        
            o	Pseudo Class :link
                Merupakan style dan kondisi default pada sebuah tag a yang memiiki href
                
            o	Pseudo Class :hover
                Merupakan style ketika kursor mouse berada diatas sebuah element
                
            o	Pseudo Class :active
                Style ketika link diklik atau klik tahan
                
            o	Pseudo Class :visited
                Style ketika sebuah link sudah pernah dikunjungi di dalam browser yang sama

        •	Pseudo Class Posisi Element
            Pseudo class ini hanya bisa digunakan apabila element memiliki parent
            
            o	Pseudo Class :first-child
                Merupakan pseudo class untuk memilih elemen pertama dari sebuah parent atau pembungkusnya
                
            o	Pseudo Class :last-child
                Merupakan pseudo class untuk memilih elemen terakhir dari sebuah parent atau pembungkusnya
                
            o	Pseudo Class :nth-child
                Merupakan pseudo class untuk memilih elemen ke-n atau bisa juga untuk ganjil(odd) dan genap(even) dari sebuah parent atau pembungkusnya bisa juga dengan pola kelipatan atau pengurangan atau pertambahan 
                
            o	Pseudo Class :first-of-type
                Memilih elemen pertama dari sebuah jenis / tipe tag
                
            o	Pseudo Class :last-of-type
                Memilih elemen terakhir dari sebuah jenis / tipe tag
