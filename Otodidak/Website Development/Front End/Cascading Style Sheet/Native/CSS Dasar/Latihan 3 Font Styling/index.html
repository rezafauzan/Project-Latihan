Catatan

        Font Styling
        Merupakan Teknik css untuk menghias font terkait typeface salah satunya adalah dengan property font-family, font-size, font-weight, font-variant

        •	Property font-family
            Yaitu property yang digunakan untuk mengatur jenis font yang akan digunakan valuenya ada dua yaitu nama font dan genericnya

            Secara default font-family nya adalah Times News Roman
            Property font family ini bisa diisi beberapa nama font dipisahkan menggunakan koma bertujuan untuk apabila di dalam sistem operasi user tidak ditemukan font family pertama akan dicoba dengan font family selanjutnya dan apabila seluruh nama font yang disertakan tidak ada maka akan menggunakan genericnya yaitu serif atau sans-serif

            Font sans-serif merupakan tipe font yang tidak memiliki kaki sedangkan serif memiliki kaki
        
        •	Property font-size
            Merupakan property untuk mengatur ukuran dari font yang akan ditampilkan nilainya bisa diisi berbagai macam css unit termasuk unit absolute,relative dll

        •	Property font-weight
            Merupakan property yang digunakan untuk menaruh bobot pada font dalam konteks ini bisa diartikan bahwa font-weight itu untuk mengatur ketebalan font value yang diisikan bisa lighter, normal, angka dari 100 hingga 900, bold dan bolder

        •	Property font-variant
            Merupakan property untuk membuat tulisan menjadi small caps yaitu membuat seluruh element yang dihias menjadi huruf capital tapi setiap huruf awal pada kata menjadi lebih besar

        •	Property font-style
            Merupakan property untuk mengubah font menjadi normal, italic dan oblique (di miringkan secara paksa apabila font family tidak memiliki style italic)

        •	Property line-height
            Merupakan property untuk mengatur jarak antar baris atau paragraf

        •	SHORTHAND FONT STYLING
            Untuk melakukan styling pada font css memiliki shorthand untuk menaruh semua property font styling menjadi satu dengan menggunakan property font
