Catatan

        Inheritance / Pewarisan
        Merupakan konsep dalam css yang dimana sebuah elemen dapat mewarisi beberapa nilai dari properti yang dimiliki oleh parentnya 
        Apabila element parent memiliki property yang bisa diwariskan diantaranya adalah property color, font, letter-spacing, line-height, list-style, text-align, text-indent, text-transform, visibility, white-space 
        
        Maka element childrennya cenderung akan memiliki nilai dari property yang sama namun ada kasus tertentu dimana property parent tidak diwariskan secara langsung contohnya pada tag anchor karena secara default tag anchor ini tidak mewarisi property color
        
        Dan untuk membuat tag seperti ini mewarisi property dari parentnya maka perlu diberikan property yang sama dengan nilai inherit
