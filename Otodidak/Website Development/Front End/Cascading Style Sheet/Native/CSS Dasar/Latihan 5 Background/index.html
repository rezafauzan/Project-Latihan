Catatan

        Background
        Merupakan cara untuk memberi background atau latar belakang pada sebuah element terdapat beberapa property terkait background yaitu :
        
        •	Property background-color
            Untuk menjadikan warna sebagai latar belakang

        •	Property background-image
            Untuk menjadikan gambar sebagai latar belakang

        •	Property background-position
            Untuk mengatur posisi gambar latar value nya bisa diisi dan dikombinasikan yaitu top, left, bottom, right ,center dan x% y% dan x-pos y-pos

        •	Property background-repeat
            Untuk mengatur jenis pengulangan dari 1 gambar pada background

            Property-property diatas bisa digunakan sekaligus pada 1 elemen

        •	SHORTHAND BACKGROUND
            CSS memiliki property shorthand untuk mengatur background yaitu dengan menggunakan property background: color url() position repeat
