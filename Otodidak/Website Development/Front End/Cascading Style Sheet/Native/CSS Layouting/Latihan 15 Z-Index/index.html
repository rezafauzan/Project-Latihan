Catatan

        Z-Index
        Normalnya urutan element HTML adalah yang paling terakhir ditulis akan berada paling depan
        Untuk mengatur tumpukan element agar bisa berada didepan atau dibelakang element lain bisa dengan cara menuliskan tag html tersebut dibawah atau diatas element lain

        Pada gambar diatas terdapat tiga kotak Kotak 1 berwarna darkblue, Kotak 2 berwarna darkgreen, Kotak 3 berwarna darkred
        Ketiga kotak tersebut memiliki property position absolute, kotak dua dan tiga diberi property top dan left sehingga menghasilkan tampilan seperti pada gambar 
        Terlihat pada gambar tersebut bahwa kotak dua menimpa kotak satu dan kotak tiga menimpa kotak dua
        Hal ini terjadi karena Urutan penulisan mark up pada source code HTML nya

        normalnya urutan element HTML adalah yang paling terakhir ditulis akan berada paling depan
        dan kotak satu berada paling pertama dan kotak tiga paling akhir di source code
        untuk mengatur urutannya bisa dengan cara memindahkan element yang di mau berada dibawah element yang di mau pada source code 

        Bisa terlihat pada gambar bahwa sekarang di source code dan hasilnya Kotak satu berada paling depan dan dua paling bawah
        Hal ini cukup merepotkan untuk mengatur urutan tumpukannya karena harus selalu mengubah HTML nya

        Z index merupakan property untuk mengatur urutan tumpukan element non static berdasar sumbu z

        Bisa dilihat sekarang markupnya ditulis berurutan tapi urutan tampilnya diatur menggunakan property z-index
        Property ini diisi dengan nilai angka yang merepresentasikan akan berada di tumpukan ke berapa semakin besar nilainya maka semakin berada di depan

        z-index juga bisa diisi nilai yang besar jika ingin selalu berada paling depan misalnya 9999 atau diisi -9999 agar element berada paling belakang
