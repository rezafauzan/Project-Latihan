Catatan

        Box Model : Border
        Border Merupakan batas di sekeliling padding dan content untuk membuat border memerlukan property css border diisi dengan tebal border, style border, dan warna border

        Property Border ini memiliki beberapa style yaitu solid, dotted, dashed, dan double

        Sama seperti padding dan margin, border juga bisa diatur hanya salah satu sisinya mau sisi atas, kanan, bawah, kiri 

        Border sama seperti padding penambahan border akan menambah ukuran dari box nya

        Untuk mengakalinya sama seperti padding bisa dengan mengurangi width dengan jumlah padding dan border atau menggunakan property box-sizing

        Box Model : Box-Sizing
        Box-Sizing merupakan Property untuk membuat ukuran kotak tetap seperti yang diisi di property width meskipun terdapat padding dan border
