Catatan

    Container Collapse
        Container Collapse merupakan kejadian apabila seluruh element dalam suatu pembungkus / container diberi float 
        
        Karena float membuat element menjadi keluar dari flownya atau Out-Of Flow dan membuat pembungkusnya dan element lain yang tidak memiliki float memanggap element float itu tidak ada
        
        Jika seluruh element dalam suatu pembungkus dianggap tidak ada maka akan menyebabkan containernya collapse atau runtuh yang dimana container akan menjadi kecil karena menganggap tidak memiliki element

        Untuk Mengatasi Container Collapse perlu untuk membersihkan atau menghentikan atau membatasi float
        Ada beberapa cara untuk melakukannya yaitu dengan property overflow, menggunakan div kosong dan menggunakan metode clear fix 

    •	Metode Clear Float Dengan Property Overflow:Auto
        Dengan menambahkan property overflow:auto pada container maka akan membuat element dengan float akan dianggap ada lagi

    •	Metode Clear Float Dengan Property Clear

        Property clear berfungsi untuk menghentikan atau membersihkan float dan merupakan best practice untuk membersihkan float
        
        Setiap akhir dari element float harus diberi div kosong setelah element float (Khusus untuk text wrapping div kosong ditambahkan setelah text agar text wrapping tetap sesuai harapan) dengan property clear dan diberi value dengan ketentuan sebagai berikut
        
        Apabila float nya left maka value clearnya juga left jika float right maka harus diberi clear:right; apabila ingin kedua nya isi clear dengan nilai both

        Namun dengan menggunakan cara seperti ini ada pendapat bahwa cara ini membuat source code jadi tidak rapih karena terdapat div kosong dan tidak bermakna

    •   Metode Clear Float Dengan Script Micro Clearfix
        Cara ini dipercaya merupakan solusi yang lebih baik untuk membersihkan atau menghentikan float yaitu dengan script dari Nicolas Gallagher
