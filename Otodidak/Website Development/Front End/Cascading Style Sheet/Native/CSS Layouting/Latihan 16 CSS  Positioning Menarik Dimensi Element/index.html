Position : Fixed

Element ini memiliki property position fixed akan selalu berada di atas titik 0 biasanya digunakan sebagai navbar

element ini juga tidak memiliki property width karena digantikan oleh left:0; right:0;

Catatan

        Position juga bisa digunakan untuk menarik agar element html memiliki ukuran contohnya jika sebuah element diberi position absolute dan diberi left 0 right 0 maka akan membuat element tersebut ditarik ke kiri dan kanan  

        Bisa dilihat element berwarna dark blue ini tidak memiliki property width tapi ukurannya ditarik ke kiri dan kanan sehingga menghasilkan ukuran yang sama seperti pada navbar
        Hal ini juga berlaku dengan property top dan bottom
        
        Dan jika diberi nilai selain 0 maka akan menambahkan jarak dan sisanya adalah element tersebut
