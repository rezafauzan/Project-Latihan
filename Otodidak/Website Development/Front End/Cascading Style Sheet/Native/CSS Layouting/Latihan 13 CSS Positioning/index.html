Catatan

        CSS Positioning
        Position digunakan ketika ingin menggeser atau menggerakan sebuah element ke suatu arah tanpa merubah atau mengganggu atau mempengaruhi element disekitarnya
        Property yang digunakan adalah position dan bisa diisi dengan 4 value yaitu static, relative, absolute, fixed
        
        •	Static (Default)  
            Static merupakan nilai default untuk position 
            
            Apabila sebuah element diberi position selain static (non-static) akan membuat element tersebut seolah olah berbeda dimensi dari element lainnya
            
            Dan element yang diberi position selain static dapat menggunakan property top, left, bottom dan right untuk mengatur posisinya
        
            Bisa diilustrasikan bahwa ketiga kotak diatas sejajar berada di dimensi yang sama

        •	Relative
            Posisi relative biasanya digunakan ketika ingin membuat suatu element maju satu dimensi atau keluar dari flownya
            Relative merupakan value non-static  dari position ketika sebuah element diberi position relative element tersebut akan maju satu dimensi keluar dari normal flownya seperti pada ilustrasi di catatan.docx

            Catatan: itu hanya ilustrasi saja element html bukanlah objek 3D 
            Ilustrasi di catatan.docx memperlihatkan bahwa element dengan position relative tampak sama seperti element dengan position static yang membedakan adalah element dengan position relative maju satu dimensi atau out-of flow  sehingga memiliki akses untuk bergerak ke arah atas bawah kanan dan kiri

            Property top, left, bottom, dan right memiliki arti bahwa akan menggerakan element kearah tersebut berlawanan sebanyak value nya misal top:30px; left:25px; bottom:20px; right:15px;

            Ketika menggerakan element dengan position relative menggunakan property top left bottom right, ruang yang ditempati element tersebut masih ada
            Disebut relative karena ketika menggerakan element dengan position relative elementnya akan bergerak relative terhadap posisi semulanya

            Jika diberi property top dan left 0px ( posisi acuan ) maka element tidak berubah posisinya
