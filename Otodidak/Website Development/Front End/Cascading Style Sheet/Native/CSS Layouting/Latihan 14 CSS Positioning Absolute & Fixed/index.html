Catatan

        •	Absolute
            Sama seperti relative position absolute membuat elementnya maju satu dimensi dan keluar dari flow bedanya adalah element lain tidak akan menganggap ada dan ruang kosong yang ditempatinya akan benar benar dianggap kosong
            
            Dan ketika element dengan position absolute digerakan element akan bergerak relative terhada posisi dari parentnya bukan ruang asalnya
            
            Gambar di catatan.docx ini element berwarna ungu diberi position absolute dan right sebesar 0 yang terjadi adalah element tersebut berada di titik 0 sebelah kanan dari object window
            
            Element ini juga sama dengan position absolute dan left 0 berada di titik 0 sebelah kiri dari parentnya yaitu body atau lebih tepatnya object window
            Ketika element dengan position absolute diberi pembungkus dengan position static dan digeserkan maka element tersebut tidak akan langsung berada di posisi yang diinginkan melainkan akan keluar dari pembungkusnya hal ini dikarenakan di dalam dimensi element yang posisinya absolute dia tidak memiliki pembungkus lihat ilustrasi pada catatan.docx

            Untuk mengatasi ini yang harus dilakukan adalah membuat pembungkusnya berada di dimensi yang sama dengan cara memberi position non static biasanya container diberi position relative
            Sehingga container akan maju satu dimensi seperti pada ilustrasi pada catatan.docx

        •	Fixed
            Element dengan position pixed ketika digerakan maka akan selalu relative terhadap object window meskipun dibungkus menggunakan element lain

            Contohnya ketika diberi property top 0 meskipun dibungkus dan discroll elementnya akan selalu berada dititik atas 0 pada viewport atau object windownya

            Position fixed biasa digunakan untuk membuat navbar, to top button, dll
