Catatan

        CSS Reset
        Merupakan teknik untuk mengatur ulang nilai nilai css default dari element yang ada pada HTML khususnya mengenai margin dan padding
        Secara default ketika membuat sebuah element pasti sudah memiliki jarak padahal belum diberi margin pada element tersebut

        Padahal element pada gambar ini diberi margin 0, hal ini terjadi  karena ada element html yang sudah mempunyai nilai margin dan padding pada kasus ini adalah element body sudah memiliki margin dan padding 
        Dan apabila body diberi margin dan padding 0 maka element kotak pada gambar sekarang tidak memiliki jarak kosong lagi

        Hal ini sudah bisa dikatakan sudah melakukan CSS Reset selain element body banyak lagi element yang memiliki nilai default margin dan padding seperti pada h1 dan p 

        CSS Reset bertujuan agar developer memiliki kendali penuh atas CSS yang dibuat
        Untuk melakukan reset sekaligus seluruh element bisa menggunakan selector universal yang disimbolkan dengan tanda Bintang “*” 
        Namun menggunakan selector universal ini juga ada yang berpendapat bahwa terlalu berat karena seluruh element yang sudah ada maupun yang tidak ada akan terkena
        Cara yang lebih efektif adalah dengan menggunakan CSS Reset dari meyerweb.com
